# 🌑 HACKNOCTURNE

> A full-stack hackathon project featuring a core backend server, a web dashboard, a demo application, and a browser extension — built with JavaScript, TypeScript, HTML, and CSS.

---

## 📁 Project Structure

```
HACKNOCTURNE-proj/
├── core-server/       # Backend server — handles core logic and APIs
├── dashboard/         # Frontend dashboard — monitoring and control UI
│   └── src/
├── demo-app/          # Demo application showcasing project features
├── extension/         # Browser extension
├── package.json
└── .gitignore
```

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/prayas-bit/HACKNOCTURNE-proj.git
cd HACKNOCTURNE-proj

# Install dependencies
npm install
```

---

## 🧩 Components

### 🖥️ Core Server
The backend service responsible for handling API requests and core business logic.

```bash
cd core-server
npm install
npm start
```

### 📊 Dashboard
A web-based frontend dashboard for visualizing data and interacting with the core server.

```bash
cd dashboard
npm install
npm start
```

### 🧪 Demo App
A standalone demo application that demonstrates the project's capabilities end-to-end.

```bash
cd demo-app
npm install
npm start
```

### 🧩 Browser Extension
A browser extension to integrate the project's functionality directly into your browser.

**To load the extension in Chrome/Edge:**
1. Go to `chrome://extensions/`
2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the `extension/` folder
